Gore's Public Garden

Gore's public garden is a wonderful atraction to visit.

This is because of its flowers and trees that make this

an amazing and beautiful place to walkthrough or rest.

This is also commonly a place where Gore hosts some events

like the christmas festival that is hosted every year with

stalls set up and a musician or preformers. This all makes

it an amazing place to visit. If you ever stop by Gore this

is a must see attraction.
